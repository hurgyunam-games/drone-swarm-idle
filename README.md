# Drone Swarm Idle

드론을 업무별로 할당해 자원을 채집/조합/성장시키는 모바일 IDLE 게임 프로젝트입니다.

## 프로젝트 개요
- 장르: Idle / Management
- 플랫폼: Android (Flutter 개발 예정)
- 컨셉: 다른 일을 하면서도 즐길 수 있는 느긋한 템포의 성장형 게임

## 핵심 플레이
- 드론을 채집/조합/기타 업무에 할당
- 업무별 드론 수에 따라 생산 효율 변화
- 자원 생산 -> 조합 -> 업그레이드의 성장 루프
- 장기 플레이를 위한 저장/불러오기 지원 예정

## 현재 상태
- 기획 문서 작성 및 시스템 정의 진행 중
- 개발 시작 전 상세 기획 고도화 단계
